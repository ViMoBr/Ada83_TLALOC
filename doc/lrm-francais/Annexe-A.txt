
                    A. Attributs de langage prédéfinis  

Cette annexe résume les définitions données ailleurs des termes prédéfinis.
attributs du langage.  

P'ADDRESS Pour un préfixe P qui désigne un objet, une unité de programme, un
                 étiquette, ou une entrée :

                 Donne l'adresse de la première des unités de stockage
                 alloué à P. Pour un sous-programme, un package, une unité de tâche ou
                 étiquette, cette valeur fait référence au code machine associé
                 avec le corps ou la déclaration correspondante. Pour une entrée
                 pour lequel une clause d'adresse a été donnée, la valeur
                 fait référence à l'interruption matérielle correspondante. Le
                 la valeur de cet attribut est du type ADDRESS défini dans
                 le package SYSTÈME. (Voir 13.7.2.)

P'AFT Pour un préfixe P qui désigne un sous-type à virgule fixe :

                 Donne le nombre de chiffres décimaux nécessaires après le
                 point pour s'adapter à la précision du sous-type P,
                 sauf si le delta du sous-type P est supérieur à 0,1, en
                 auquel cas l'attribut donne la valeur un. (P'AFT est
                 le plus petit entier positif N pour lequel (10**N)*P'DELTA
                 est supérieur ou égal à un.) La valeur de ce
                 l'attribut est du type universal_integer. (Voir
                 3.5.10.)

P'BASE Pour un préfixe P qui désigne un type ou un sous-type :

                 Cet attribut désigne le type de base de P. Il s'agit uniquement
                 autorisé comme préfixe du nom d'un autre attribut :
                 par exemple, P'BASE'FIRST. (Voir 3.3.3.)

P'CALLABLE Pour un préfixe P approprié à un type de tâche :  

                 Renvoie la valeur FALSE lors de l'exécution de la tâche P
                 est soit terminée, soit terminée, ou lorsque la tâche est
                 anormal; sinon, renvoie la valeur TRUE. La valeur de
                 cet attribut est du type prédéfini BOOLEAN. (Voir
                 9.9.)

P'CONSTRAINED Pour un préfixe P qui désigne un objet d'un type avec
                 discriminants:   

                 Renvoie la valeur TRUE si une contrainte discriminante
                 s'applique à l'objet P, ou si l'objet est une constante
                 (y compris un paramètre formel ou un paramètre formel générique
                 du mode en); sinon, renvoie la valeur FALSE. Si P est
                 un paramètre formel générique de mode in out, ou si P est un
                 paramètre formel du mode in out ou out et la marque de type
                 donné dans la spécification du paramètre correspondant
                 désigne un type sans contrainte avec des discriminants, alors
                 la valeur de cet attribut est obtenue à partir de celle du
                 paramètre réel correspondant. La valeur de ceci
                 L'attribut est du type prédéfini BOOLEAN. (Voir
                 3.7.4.)                                            

P'CONSTRAINED Pour un préfixe P qui désigne un type ou un sous-type privé :

                 Donne la valeur FALSE si P désigne un
                 type privé non formel avec discriminants ; donne également
                 la valeur FALSE si P désigne un privé formel générique  
                 type et le sous-type réel associé est soit un
                 type sans contrainte avec discriminants ou un type sans contrainte
                 type de tableau ; sinon, renvoie la valeur TRUE. La valeur
                 de cet attribut est du type prédéfini BOOLEAN.
                 (Voir 7.4.2.)

P'COUNT Pour un préfixe P qui désigne une entrée d'une unité de tâche :

                 Donne le nombre d'appels d'entrée actuellement en file d'attente sur le
                 entrée (si l'attribut est évalué dans un accept
                 instruction pour l'entrée P, le décompte n'inclut pas le
                 tâche d'appel). La valeur de cet attribut est de
                 tapez universal_integer. (Voir 9.9.)

P'DELTA Pour un préfixe P qui désigne un sous-type à virgule fixe :

                 Donne la valeur du delta spécifié dans le paramètre fixe
                 définition de précision pour le sous-type P. La valeur de cette
                 l'attribut est du type universal_real. (Voir 3.5.10.)

P'DIGITS Pour un préfixe P qui désigne un sous-type à virgule flottante :

                 Donne le nombre de chiffres décimaux dans la décimale
                 mantisse des numéros de modèle du sous-type P. (Ceci
                 l'attribut donne le numéro D de la section 3.5.7.) L'attribut
                 la valeur de cet attribut est du type universal_integer.
                 (Voir 3.5.8.)

P'EMAX Pour un préfixe P qui désigne un sous-type à virgule flottante :

                 Donne la plus grande valeur d'exposant dans le canonique binaire
                 forme de numéros de modèle du sous-type P. (Cet attribut
                 donne le produit 4*B de la section 3.5.7.) La valeur de
                 cet attribut est du type universal_integer. (Voir
                 3.5.8.)   

P'EPSILON Pour un préfixe P qui désigne un sous-type à virgule flottante :

                 Donne la valeur absolue de la différence entre le
                 numéro de modèle 1.0 et le numéro de modèle suivant ci-dessus, pour le
                 sous-type P. La valeur de cet attribut est du type
                 universal_real. (Voir 3.5.8.)

P'FIRST Pour un préfixe P qui désigne un type scalaire ou un sous-type
                 de type scalaire :

                 Donne la limite inférieure de P. La valeur de cet attribut
                 a le même type que P. (Voir 3.5.)  

P'FIRST Pour un préfixe P approprié pour un type de tableau, ou
                 qui désigne un sous-type de tableau contraint :

                 Donne la limite inférieure de la première plage d'indice. Le
                 la valeur de cet attribut a le même type que cet attribut inférieur
                 lié. (Voir 3.6.2 et 3.8.2.)                                                  

P'FIRST(N) Pour un préfixe P approprié pour un type de tableau, ou
                 qui désigne un sous-type de tableau contraint :

                 Donne la limite inférieure de la N-ième plage d’indice. Le
                 la valeur de cet attribut a le même type que cet attribut inférieur
                 lié. L'argument N doit être une expression statique de
                 tapez universal_integer. La valeur de N doit être positive
                 (différent de zéro) et pas supérieur à la dimensionnalité du
                 tableau. (Voir 3.6.2 et 3.8.2.)

P'FIRST_BIT Pour un préfixe P qui désigne un composant d'un enregistrement  
                 objet:

                 Donne le décalage, à partir du début du premier des
                 unités de stockage occupées par le composant, du premier bit
                 occupé par le composant. Ce décalage est mesuré en
                 morceaux. La valeur de cet attribut est du type
                 entier_universel. (Voir 13.7.2.) 

P'FORE Pour un préfixe P qui désigne un sous-type à virgule fixe :

                 Donne le nombre minimum de caractères requis pour le
                 partie entière de la représentation décimale d'une valeur
                 du sous-type P, en supposant que la représentation fait
                 n'inclut pas d'exposant, mais inclut un caractère
                 préfixe qui est soit un signe moins, soit un espace. (Ce
                 le nombre minimum ne comprend pas de zéros superflus ou
                 souligne, et vaut au moins deux.) La valeur de ceci
                 l'attribut est du type universal_integer. (Voir
                 3.5.10.)

P'IMAGE Pour un préfixe P qui désigne un type ou sous-type discret :

                 Cet attribut est une fonction avec un seul paramètre.
                 Le paramètre X réel doit être une valeur du type de base
                 de P. Le type de résultat est le type prédéfini STRING.
                 Le résultat est l’image de la valeur de X, c’est-à-dire un
                 séquence de caractères représentant la valeur affichée
                 formulaire. L'image d'une valeur entière est la correspondante
                 littéral décimal ; sans soulignement, sans zéros non significatifs,
                 exposant ou espaces de fin ; mais avec un seul personnage
                 préfixe qui est soit un signe moins, soit un espace.

                 L'image d'une valeur d'énumération est soit la
                 identifiant correspondant en majuscule ou le
                 caractère littéral correspondant (y compris les deux
                 apostrophes); ni les espaces de début ni de fin ne sont
                 inclus. L'image d'un personnage autre qu'un graphique
                 le caractère est défini par l’implémentation. (Voir 3.5.5.)

P'LARGE Pour un préfixe P qui désigne un sous-type réel :

                 L'attribut donne le plus grand numéro de modèle positif de
                 le sous-type P. La valeur de cet attribut est du 
                 tapez universal_real. (Voir 3.5.8 et 3.5.10.)

P'LAST Pour un préfixe P qui désigne un type scalaire ou un sous-type
                 de type scalaire :

                 Donne la limite supérieure de P. La valeur de cet attribut
                 a le même type que P. (Voir 3.5.)

P'LAST Pour un préfixe P approprié à un type de tableau, ou
                 qui désigne un sous-type de tableau contraint :

                 Donne la limite supérieure de la première plage d'index. Le
                 la valeur de cet attribut a le même type que cette valeur supérieure
                 lié. (Voir 3.6.2 et 3.8.2.)                                          

P'LAST(N) Pour un préfixe P approprié pour un type de tableau,
                 ou qui désigne un sous-type de tableau contraint :

                 Donne la limite supérieure de la N-ème plage d’indice. Le
                 la valeur de cet attribut a le même type que cette valeur supérieure
                 lié. L'argument N doit être une expression statique de
                 tapez universal_integer. La valeur de N doit être positive
                 (différent de zéro) et pas supérieur à la dimensionnalité du
                 tableau. (Voir 3.6.2 et 3.8.2.)  

P'LAST_BIT Pour un préfixe P qui désigne un composant d'un enregistrement
                 objet:

                 Donne le décalage, à partir du début du premier des
                 unités de stockage occupées par le composant, du dernier bit
                 occupé par le composant. Ce décalage est mesuré en
                 morceaux. La valeur de cet attribut est du type
                 entier_universel. (Voir 13.7.2.)

P'LENGTH Pour un préfixe P approprié à un type de tableau,
                 ou qui désigne un sous-type de tableau contraint :

                 Renvoie le nombre de valeurs de la première plage d'index
                 (zéro pour une plage nulle). La valeur de ceci
                 l'attribut est du type universal_integer. (Voir
                 3.6.2.)

P'LENGTH(N) Pour un préfixe P approprié pour un tableau
                      type, ou qui désigne un sous-type de tableau contraint :

                      Donne le nombre de valeurs de la N-ème plage d'index
                      (zéro pour une plage nulle). La valeur de ceci
                      l'attribut est du type universal_integer. Le
                      l'argument N doit être une expression statique de type
                      entier_universel. La valeur de N doit être positive
                      (différent de zéro) et pas supérieur à la dimensionnalité de
                      le tableau. (Voir 3.6.2 et 3.8.2.)

P'MACHINE_EMAX Pour un préfixe P qui désigne un type à virgule flottante ou
                      sous-type :

                      Donne la plus grande valeur d'exposant pour la machine
                      représentation du type de base de P. La valeur de
                      cet attribut est du type universal_integer.
                      (Voir 13.7.3.)

P'MACHINE_EMIN Pour un préfixe P qui désigne un type à virgule flottante ou
                      sous-type :

                      Donne la valeur la plus petite (la plus négative) de
                      exposant pour la représentation machine de la base
                      type de P. La valeur de cet attribut est du type
                      tapez universal_integer. (Voir 13.7.3.) 

P'MACHINE_MANTISSA Pour un préfixe P qui désigne un type à virgule flottante ou
                      sous-type :

                      Donne le nombre de chiffres dans la mantisse pour le
                      représentation machine du type de base de P (le
                      les chiffres sont des chiffres étendus compris entre 0 et
                      P'MACHINE_RADIX - 1). La valeur de cet attribut
                      est du type universal_integer. (Voir 13.7.3.)                                                

P'MACHINE_OVERFLOWS Pour un préfixe P qui désigne un type ou sous-type réel :

                      Renvoie la valeur TRUE si chaque opération prédéfinie
                      sur les valeurs du type de base de P soit fournit un
                      résultat correct, ou lève l'exception
                      NUMERIC_ERROR dans les situations de débordement ; donne le
                      sinon, valeur FAUX. La valeur de cet attribut
                      est du type prédéfini BOOLEAN. (Voir 13.7.3.)

P'MACHINE_RADIX Pour un préfixe P qui désigne un type à virgule flottante ou
                      sous-type :

                      Donne la valeur de la base utilisée par la machine
                      représentation du type de base de P. La valeur de
                      cet attribut est du type universal_integer.
                      (Voir 13.7.3.) 

P'MACHINE_ROUNDS Pour un préfixe P qui désigne un type ou sous-type réel :

                      Donne la valeur TRUE si chaque arithmétique prédéfinie
                      opération sur les valeurs du type de base de P soit
                      renvoie un résultat exact ou effectue un arrondi ;
                      sinon, renvoie la valeur FALSE. La valeur de ceci
                      L'attribut est du type prédéfini BOOLEAN. (Voir
                      13.7.3.)

P'MANTISSA Pour un préfixe P qui désigne un sous-type réel :

                      Donne le nombre de chiffres binaires dans le binaire
                      mantisse des numéros de modèle du sous-type P. (Ceci
                      L'attribut donne le numéro B de la section 3.5.7 pour un
                      type à virgule flottante, ou de la section 3.5.9 pour un nombre fixe
                      type de point.) La valeur de cet attribut est du type
                      tapez universal_integer. (Voir 3.5.8 et 3.5.10.)

P'POS Pour un préfixe P qui désigne un type discret ou
                      sous-type :

                      Cet attribut est une fonction avec un seul
                      paramètre. Le paramètre X réel doit être une valeur
                      du type de base de P. Le type de résultat est le type
                      entier_universel. Le résultat est la position 
                      numéro de la valeur du paramètre réel. (Voir
                      3.5.5.)

P'POSITION Pour un préfixe P qui désigne un composant d'un enregistrement
                      objet:

                      Donne le décalage, à partir du début du premier
                      unité de stockage occupée par le dossier, du premier des
                      les unités de stockage occupées par le composant. Ce
                      le décalage est mesuré en unités de stockage. La valeur de
                      cet attribut est du type universal_integer.
                      (Voir 13.7.2.) 

P'PRED Pour un préfixe P qui désigne un type discret ou
                      sous-type :

                      Cet attribut est une fonction avec un seul
                      paramètre. Le paramètre X réel doit être une valeur
                      du type de base de P. Le type de résultat est le type de base
                      type de P. Le résultat est la valeur dont la position
                      le nombre est inférieur de un à celui de X. L'exception
                      CONSTRAINT_ERROR est déclenché si X est égal à P'BASE'FIRST.
                      (Voir 3.5.5.)

P'RANGE Pour un préfixe P approprié pour un tableau
                      type, ou qui désigne un sous-type de tableau contraint :

                      Donne la première plage d'indices de P, c'est-à-dire le
                      gamme P'FIRST .. P'LAST. (Voir 3.6.2.)                                        

P'RANGE(N) Pour un préfixe P approprié pour un tableau
                      type, ou qui désigne un sous-type de tableau contraint :

                      Donne la N-ème plage d'indice de P, c'est-à-dire la plage
                      P'PREMIER(N) .. P'LAST(N). (Voir 3.6.2.)

P'SAFE_EMAX Pour un préfixe P qui désigne un type à virgule flottante ou
                      sous-type :

                      Donne la plus grande valeur d'exposant dans le binaire
                      forme canonique des nombres sûrs du type de base de
                      P. (Cet attribut donne le numéro E de la section
                      3.5.7.) La valeur de cet attribut est du type
                      entier_universel. (Voir 3.5.8.) 

P'SAFE_LARGE Pour un préfixe P qui désigne un type ou sous-type réel :

                      Donne le plus grand nombre positif sûr de la base
                      type de P. La valeur de cet attribut est du type
                      tapez universal_real. (Voir 3.5.8 et 3.5.10.)

P'SAFE_SMALL Pour un préfixe P qui désigne un type ou sous-type réel :

                      Donne le plus petit nombre sûr positif (différent de zéro)
                      du type de base de P. La valeur de cet attribut
                      est du type universal_real. (Voir 3.5.8 et
                      3.5.10.)

P'SIZE Pour un préfixe P qui désigne un objet :

                      Donne le nombre de bits alloués pour contenir le
                      objet. La valeur de cet attribut est du type
                      entier_universel. (Voir 13.7.2.)

P'SIZE Pour un préfixe P qui désigne n'importe quel type ou sous-type :

                      Donne le nombre minimum de bits requis par
                      la mise en œuvre pour détenir tout objet éventuel de
                      le type ou sous-type P. La valeur de cet attribut
                      est du type universal_integer. (Voir 13.7.2.)

P'SMALL Pour un préfixe P qui désigne un sous-type réel :

                      Donne le plus petit numéro de modèle positif (différent de zéro)
                      du sous-type P. La valeur de cet attribut est de
                      le type universal_real. (Voir 3.5.8 et 3.5.10.)

P'STORAGE_SIZE Pour un préfixe P qui désigne un type d'accès ou
                      sous-type :

                      Donne le nombre total d'unités de stockage réservées
                      pour la collection associée au type de base de
                      P. La valeur de cet attribut est du type
                      entier_universel. (Voir 13.7.2.) 

P'STORAGE_SIZE Pour un préfixe P qui désigne un type de tâche ou une tâche
                      objet:

                      Donne le nombre d'unités de stockage réservées pour chacun
                      activation d'une tâche de type P ou pour le
                      activation de l'objet tâche P. La valeur de ce
                      l'attribut est du type universal_integer. (Voir
                      13.7.2.)                                                      

P'SUCC Pour un préfixe P qui désigne un type discret ou
                      sous-type :

                      Cet attribut est une fonction avec un seul
                      paramètre. Le paramètre X réel doit être une valeur
                      du type de base de P. Le type de résultat est le type de base 
                      type de P. Le résultat est la valeur dont la position
                      nombre est supérieur de un à celui de X. Le
                      l'exception CONSTRAINT_ERROR est levée si X est égal à
                      P'BASE'LAST. (Voir 3.5.5.)

P'TERMINATED Pour un préfixe P approprié à un type de tâche :

                      Renvoie la valeur TRUE si la tâche P est terminée ;
                      sinon, renvoie la valeur FALSE. La valeur de ceci
                      L'attribut est du type prédéfini BOOLEAN. (Voir
                      9.9.) 

P'VAL Pour un préfixe P qui désigne un type discret ou
                      sous-type :

                      Cet attribut est une fonction spéciale avec un seul
                      paramètre X qui peut être de n’importe quel type entier. Le
                      le type de résultat est le type de base de P. Le résultat est
                      la valeur dont le numéro de position est le
                      valeur universal_integer correspondant à X. La
                      l'exception CONSTRAINT_ERROR est levée si le
                      La valeur universal_integer correspondant à X n'est pas dans
                      la plage P'POS(P'BASE'FIRST) .. P'POS(P'BASE'LAST).
                      (Voir 3.5.5.)

P'VALUE Pour un préfixe P qui désigne un type discret ou
                      sous-type :

                      Cet attribut est une fonction avec un seul
                      paramètre. Le paramètre X réel doit être une valeur
                      du type prédéfini STRING. Le type de résultat est
                      le type de base de P. Tout début et tout début
                      espaces de la séquence de caractères qui
                      correspond à X sont ignorés.   

                      Pour un type énumération, si la séquence de
                      caractères a la syntaxe d'un littéral d'énumération
                      et si ce littéral existe pour le type de base de P,
                      le résultat est la valeur d'énumération correspondante.
                      Pour un type entier, si la séquence de caractères
                      a la syntaxe d'un littéral entier, avec un
                      caractère de début unique facultatif qui est un plus ou
                      signe moins, et s'il y a une valeur correspondante dans
                      le type de base de P, le résultat est cette valeur. Dans
                      dans tous les autres cas, l'exception CONSTRAINT_ERROR est
                      soulevé. (Voir 3.5.5.)   

P'WIDTH Pour un préfixe P qui désigne un sous-type discret :

                      Donne la longueur maximale de l'image sur toutes les valeurs de
                      le sous-type P (l'image est la séquence de
                      caractères renvoyés par l'attribut IMAGE). Le
                      la valeur de cet attribut est du type
                      entier_universel. (Voir 3.5.5.)
